# 📊 DecodeLabs AI Project 2 – KNN Classification Pipeline

A complete supervised machine learning pipeline built using the **K-Nearest Neighbors (KNN)** algorithm on the Iris dataset. The project demonstrates the end-to-end workflow from data preprocessing to model evaluation.

Developed during my AI Internship at DecodeLabs.

---

## 🚀 Features

- Load the Iris dataset
- Train/Test split with stratification
- Feature scaling using StandardScaler
- Hyperparameter tuning using the Elbow Method
- KNN model training
- Classification Report
- Confusion Matrix visualization
- Modular and production-style code structure

---
